# 💢 Deal or Not Deal - 🕹️ PC Game (C#)

>A C# implementation of the classic “Deal or No Deal” game, created during my college course to demonstrate programming fundamentals and game logic.
>
>Created from scratch, I managed to create images for the game such as the background and boxes, also design the layout based on images I saw online. This was an individual project which I enjoyed a lot as it was the first game I actually saw to come to live create for myself.
>
>Game can actually be easily played as long as you have .NET SDK or Runtime installed in your computer. If so, it should be Download and Play 🎮
